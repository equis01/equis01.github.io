Aviso de Privacidad

Este aviso aplica al formulario de investigación de mercado sobre cuidado animal disponible en este sitio. Su finalidad es explicar qué datos se recopilan, para qué se usan y cómo puedes ejercer tus derechos.

Responsable

El responsable del tratamiento de los datos recopilados mediante este formulario es Eduardo Vázquez, como promotor del estudio y posible desarrollo de una plataforma tecnológica para el ecosistema de cuidado animal.

Contacto para temas de privacidad: luis@evamtz.me

Datos que se pueden recopilar

  • Respuestas del formulario sobre necesidades, problemas, experiencias, operación, adopción, ventas, rescate o cuidado animal.
  • Correo electrónico y WhatsApp, si decides proporcionarlos.
  • IP, ciudad y país aproximados, obtenidos mediante un servicio de geolocalización aproximada por IP.
  • Datos técnicos mínimos del envío necesarios para seguridad, prevención de abuso y análisis básico.

Finalidades principales

  • Realizar investigación de mercado y validar dolores reales de posibles usuarios o clientes.
  • Diseñar, priorizar y mejorar una posible plataforma tecnológica comercial relacionada con cuidado animal.
  • Contactarte si autorizaste dejar correo o WhatsApp, por ejemplo para aclarar respuestas, invitarte a entrevistas o compartir avances del proyecto.
  • Prevenir spam, abuso, respuestas duplicadas o uso automatizado del formulario.

Finalidades secundarias

Con tu autorización, las respuestas podrán usarse de forma agregada, anonimizada o editada para reportes, presentaciones, validación del proyecto, materiales de negocio o comunicación pública. No se publicarán datos de contacto personales sin consentimiento específico.

Transferencias y terceros

El formulario puede apoyarse en servicios tecnológicos externos para recibir respuestas, detectar ubicación aproximada por IP, alojar la página o procesar datos operativos. Estos servicios solo deben usarse para las finalidades descritas y no para vender tus datos personales.

Derechos ARCO y revocación

Puedes solicitar acceso, rectificación, cancelación u oposición respecto a tus datos personales, así como revocar tu consentimiento, escribiendo al correo de contacto. Para ubicar tu información, incluye el correo usado en el formulario y una descripción clara de tu solicitud.

Conservación

Los datos se conservarán durante el tiempo necesario para analizar el mercado, validar el proyecto, dar seguimiento a participantes y cumplir obligaciones aplicables. Posteriormente podrán eliminarse, anonimizarse o conservarse solo en forma agregada.

Cambios al aviso

Este aviso puede actualizarse conforme avance el proyecto, cambien los servicios utilizados o se definan nuevas funcionalidades. La versión vigente estará disponible en esta página.

Nota importante

Este documento es una base informativa para una etapa temprana.
