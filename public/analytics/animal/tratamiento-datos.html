Tratamiento de Datos

Esta página resume de forma práctica cómo se tratan los datos del formulario. La intención es recopilar señales de mercado para entender qué problemas existen, qué tan urgentes son y qué tipo de solución comercial podría tener sentido.

Resumen operativo

Dato Uso Motivo
Respuestas sobre experiencia, operación, costos, adopción, ventas o rescate Análisis cualitativo y cuantitativo Detectar dolores, patrones, disposición de uso y oportunidades de producto
Correo electrónico Contacto posterior si la persona lo proporciona Entrevistas, aclaraciones, validación de hipótesis o comunicación del proyecto
WhatsApp Contacto opcional Seguimiento más directo si la persona lo autoriza
IP, ciudad y país aproximados Contexto geográfico y seguridad Entender mercados, evitar abuso y filtrar respuestas duplicadas o automatizadas
Preferencias de privacidad indicadas en el formulario Control de uso público o privado de respuestas Respetar si las respuestas pueden compartirse públicamente o solo usarse de forma confidencial

Uso público de respuestas

Si autorizas compartir información públicamente, se procurará usarla de forma agregada, resumida, anonimizada o editada para evitar identificarte directamente. Si no autorizas uso público, tus respuestas se usarán solo para análisis interno del proyecto.

Datos sensibles

El formulario no busca recopilar datos personales sensibles. Evita incluir información médica humana, documentos oficiales, domicilios completos, datos bancarios o cualquier dato que no sea necesario para explicar tu experiencia.

Seguridad

Se aplicarán medidas razonables para limitar el acceso a las respuestas y reducir exposición innecesaria. Al estar en etapa temprana, el sistema puede apoyarse en herramientas externas de formularios, alojamiento, automatización o análisis.
