Aviso de Privacidad

Este aviso corresponde al formulario de investigación de mercado sobre viajes, tours, reservas, pagos y organización de grupos. Su objetivo es explicar qué datos se recopilan y cómo se usan.

Responsable

El responsable del tratamiento de los datos recopilados mediante este formulario es Eduardo Vázquez, como promotor del estudio y posible desarrollo de una plataforma tecnológica para experiencias, tours y viajes organizados.

Contacto para privacidad: luis@evamtz.me

Datos que se recopilan

  • Rol principal: viajero u organizador.
  • Género y rango de edad.
  • Respuestas sobre dolores al reservar, gestionar grupos, pagos, confianza, precios y uso potencial de herramientas digitales.
  • IP aproximada y user agent para prevención de abuso, rate limit y seguridad básica.

Finalidades

  • Entender dolores reales de viajeros y organizadores.
  • Validar hipótesis de producto, confianza, pagos, abonos, reservas y gestión operativa.
  • Evaluar si existe oportunidad para una plataforma comercial de viajes y tours.
  • Prevenir spam, respuestas automatizadas o abuso del formulario.

Uso de resultados

Las respuestas podrán analizarse de forma individual o agregada para tomar decisiones de producto. Si se usan en reportes, presentaciones o materiales del proyecto, se procurará hacerlo de forma anónima o agregada.

Transferencias y terceros

El formulario puede utilizar servicios externos de alojamiento, Google Apps Script, Google Sheets y servicios de IP para operar el envío, almacenamiento y seguridad básica.

Derechos ARCO

Puedes solicitar acceso, rectificación, cancelación u oposición de tus datos, o revocar tu consentimiento, escribiendo al correo de contacto. Incluye suficiente información para ubicar tu respuesta.

Nota

Este aviso es una base para una etapa de investigación de mercado. Antes de lanzar una plataforma comercial con cuentas, pagos o datos más sensibles, deberá revisarse y actualizarse legalmente.
